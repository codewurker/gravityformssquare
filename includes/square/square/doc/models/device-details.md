## Device Details

Details about the device that took the payment.

### Structure

`DeviceDetails`

### Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `deviceId` | `?string` | Optional | Square-issued ID of the device. |
| `deviceName` | `?string` | Optional | The name of the device set by the merchant. |

### Example (as JSON)

```json
{
  "device_id": null,
  "device_name": null
}
```
